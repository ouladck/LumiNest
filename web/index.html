Your macOS media gallery, built for speed.

LumiNest helps you browse large photo/video folders, preview instantly, organize with albums/favorites, and navigate with keyboard shortcuts.

Features

Built for real-world media libraries, including very large folders.

Library & Navigation

Folder scan, grid/list views, quick search, media-type filtering, sorting, and next/previous preview navigation.

Viewer

Fullscreen preview, side arrows, swipe navigation, metadata panel, image prefetching, and stable video playback.

Organization

Favorites, custom albums, multi-selection, and context actions (Reveal in Finder, Delete, Copy path).

Settings & i18n

Persistent defaults, configurable root folder, viewer preferences, and multilingual UI support.

Install

Choose Homebrew for end users, or build from source for development.

Homebrew

brew tap ouladck/luminest brew install --cask luminest

Build from source

git clone https://github.com/ouladck/LumiNest.git cd LumiNest open Package.swift

Keyboard Shortcuts

CmdO Select folder

CmdF Search

Cmd, Open settings

Space Play/Pause video

R Replay video

Esc Close viewer
